# Екатерина Князева

Одностраничный памятный сайт-таймлайн для Екатерины Князевой.

Проект полностью статический: HTML, CSS, JavaScript и локальные медиафайлы. Сборка не нужна.

## Что внутри

- `index.html` — вся разметка, стили и логика сайта.
- `photos/` — фотографии и видео для таймлайна.
- `paterns/` — фирменные паттерны для фона и карточек.
- `fonts/` — локальные шрифты Tavrida.
- `img/` — логотипы, включая favicon.

## Запуск локально

Можно просто открыть `index.html` в браузере.

Если нужен локальный сервер:

```bash
python3 -m http.server 8000
```

После запуска открыть:

```text
http://localhost:8000
```

## Деплой

Проект можно разместить на любом статическом хостинге:

- GitHub Pages
- Netlify
- Vercel
- Timeweb / Beget / любой обычный хостинг

Для деплоя достаточно загрузить содержимое папки проекта целиком, сохранив структуру директорий.

## Как редактировать таймлайн

Все события находятся в массиве `moments` внутри `index.html`.

Каждый блок выглядит так:

```js
{
  title: "Название момента",
  text: "Описание момента.",
  media: "photos/example.jpeg",
  type: "image",
  tone: "sky",
  pattern: "8",
  tilt: "1deg"
}
```

Для видео используется:

```js
type: "video"
```

## Загрузка сайта

Перед показом сайта включается экран загрузки с текстом “Надо немного подготовиться”.

Он ждёт загрузку:

- шрифтов;
- логотипов;
- паттернов;
- изображений;
- metadata видео.

Видео целиком не блокирует открытие сайта, чтобы страница не зависала на медленном интернете.
